# Portfolio Website

A clean and minimalist portfolio website for showcasing design projects.

## Files Overview

- `index.html` - Home page with project grid
- `project.html` - Project detail page template
- `about.html` - About page
- `contact.html` - Contact page
- `styles.css` - Main stylesheet
- `project-styles.css` - Project page specific styles
- `project.js` - JavaScript for project page content
- `images/` - Folder containing logo and example images

## How to Use

1. **View the website**: Open `index.html` in your web browser

2. **Add your project images**:
   - Replace the black placeholder boxes with your actual project images
   - Add your images to the `images/` folder
   - Update the HTML to point to your images

3. **Customize project information**:
   - Edit `project.js` to update project titles, descriptions, and categories
   - Each project has: category, title, and description

4. **Update About and Contact pages**:
   - Edit `about.html` to add your bio
   - Edit `contact.html` to add your contact information

5. **Add or remove projects**:
   - In `index.html`, add/remove project items in the grid
   - In `project.js`, add/remove project entries
   - Make sure to update the sidebar menu on all pages

## Customization Tips

- To change colors, edit `styles.css`
- To change fonts, modify the `font-family` in `styles.css`
- To adjust spacing, modify padding and gap values in the CSS files
- The layout is responsive and works on mobile devices

## Publishing Your Website

To make your website live on the internet:

1. **Using GitHub Pages** (Free):
   - Create a GitHub account
   - Create a new repository
   - Upload all files
   - Enable GitHub Pages in repository settings

2. **Using Netlify** (Free):
   - Go to netlify.com
   - Drag and drop your folder
   - Your site will be live instantly

3. **Using other hosting services**:
   - Upload all files to any web hosting service
   - Make sure `index.html` is in the root directory
